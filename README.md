# mcoffline

Convert minecraft whitelist to offline-mode whitelist.

# Usage
```
mcoffline k0taa
mcoffline whitelist.json
mcoffline < whitelist.json > whitelist-offline.json
```

# Install
```
make all
sudo make install
```

# Uninstall
```
sudo make uninstall
```
